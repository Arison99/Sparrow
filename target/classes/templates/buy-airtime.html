API Information

REST Endpoint

POST /admin/buyAirtime

Parameters:

  • phone (string, required)
  • amount (number, required)

Example cURL

curl -X POST http://localhost:8080/admin/buyAirtime \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "phone=+1234567890&amount=20.00"
